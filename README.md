# Daycore(<https://daycore.co.kr>)
> 데이코어 주식회사 회사소개 홈페이지입니다.

# 개발 환경
* Vue
* bulma

# 설치 방법
```bash
# 의존성 라이브러리 설치
npm install

# 로컬 환경에서 실행
npm run dev

# 프로덕션 상태로 빌드
npm run build
```
